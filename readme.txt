== Installation ==
This section describes how to install the plugin and get it working.

1. Install through your wordpress panel. Simply Upload and install the zip file to install.

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Go to your Widgets panel and simply drag the loan calculator widget to the sidebar to display it.

4. Or Add this short code [loan-calculator] to any page where you want to show the loan calculator.
